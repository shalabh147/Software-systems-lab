◆ printSpiral()

void printSpiral ( int  myArr[7][7])

This function prints an array by going through it in a spiral fashion.

That is, it starts at the top-left corner and prints the first column in a line.
Then it turns left and prints the last row (6 elements as the first has already been printed).
Then it turns left again and prints the last column (again 6 elements).
Turns left again and print the remaining 5 elements from the first row.
It caries on like this till the whole array is printed.
