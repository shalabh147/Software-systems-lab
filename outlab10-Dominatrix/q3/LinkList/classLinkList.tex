\doxysubsection{Detailed Description}
A class implementing a Deque using a doubly linked list. ~\newline
 Deque (De\+Queue) stands for Double-\/ended Queue. ~\newline
It is just like a queue but does not support F\+I\+FO structure. ~\newline
 Insertion and deletion can be done from both side( F\+R\+O\+N\+T \& R\+E\+A\+R). ~\newline
 Implementation uses the Node class. Data members are a couple of nodes to store the data at the front and rear of the deque. Member functions have been documentated.
